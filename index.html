Face the force con html5 API

View on GitHub Download .zip Download .tar.gz

Como puedes ver... Madrid ha sido invadida por cascos gigantes de Star Wars! Estarán en la ciudad las próximas semanas (30/10/2015-30/11/2015).

La idea de este repositorio es hacer usar el poder de HTML5 API para crear ejemplos y aplicaciones interactivas usando como excusa los ocho cascos gigantes repartidos por toda la ciudad.

Geolocalización con Google Maps

Usuario y los cascos en un mapa. Ver demo!

Geolocalización y distancia relativa

Usuario y los cascos en un mapa incluida la distancia relativa entre el usuario y el casco (en línea recta). Ver demo!
